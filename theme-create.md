# 🎨 Кастомные Темы для md.web (GS26)

Добро пожаловать в репозиторий кастомных тем для Markdown-редактора `md.web`. Здесь вы можете создать свою уникальную тему для интерфейса стиля **GS26**, опубликовать её и поделиться с сообществом!

## 🚀 Как создать и опубликовать свою тему:

1. **Сделайте Форк или Скопируйте шаблон**:
   Возьмите код из файла `theme.css` в этом репозитории за основу.
   
2. **Отредактируйте CSS под свой стиль**:
   * Меняйте переменные, фоновые цвета (`background-color`), цвета шрифтов и градиенты.
   * Добавляйте свои `@keyframes` анимации для панелей `.gs-panel` или кнопок.
   * *Важно:* Изменяйте только те классы, которые относятся к `.theme-gs26`, чтобы не сломать ретро-режим `FIE11`.

3. **Загрузите файл на свой GitHub**:
   Создайте публичный репозиторий (например, `my-md-theme`) и загрузите туда ваш измененный файл `theme.css`.

4. **Получите RAW ссылку**:
   Откройте ваш `theme.css` на GitHub, нажмите кнопку **"Raw"** в правом верхнем углу кода и скопируйте URL из адресной строки браузера. Ссылка должна выглядеть примерно так:
   `[https://raw.githubusercontent.com/ваш-юзернейм/имя-репозитория/main/theme.css](https://raw.githubusercontent.com/ваш-юзернейм/имя-репозитория/main/theme.css)`

5. **Добавьте в приложение**:
   Перейдите в интерфейс `md.web`, нажмите **«Магазин тем»**, вставьте полученную RAW ссылку в текстовое поле и нажмите **«Добавить»**. Тема автоматически применится и сохранится в вашем `localStorage`.

## 📢 Как предложить тему для добавления в официальный Магазин?

Если вы хотите, чтобы ваша тема отображалась в официальном списке магазина (с уникальной SVG иконкой), создайте Issue в основном репозитории проекта по ссылке:
👉 [https://github.com/JunDevX/md.web/issues](https://github.com/JunDevX/md.web/issues)

В Issue укажите:
* Название темы.
* Ссылку на ваш репозиторий с темой.
* Ссылку на RAW CSS файл.
